## 1. 김용 소설의 이야기 구조

### 1.1 공통 구조 공식

김용 소설의 기본 패턴 (출처: [Wuxia Wikipedia](https://en.wikipedia.org/wiki/Wuxia), [Jin Yong's Four Pillars](https://hongkonger.world/2024/06/25/jin-yongs-four-pillars-of-wuxia-critique/)):

```
[비극적 사건/출생의 비밀] → [고난 속 성장] → [기연 획득] → [세력 간 갈등에 투입]
→ [연속 기연으로 급성장] → [정사대전/최종 대결] → [은거/비극적 결말]
```

김용이 제시한 무협소설 4대 평가 기준:
1. **주제(Theme)** - 협의 정신, 가치관
2. **인물(Characterisation)** - 입체적 다차원 캐릭터
3. **플롯과 구조(Plot & Structure)** - 치밀한 인과관계
4. **배경(Setting)** - 역사+강호 세계관

### 1.2 작품별 구조 분석

#### 사조영웅전 (유가적 협객 - 곽정)

```
구조: 정통 영웅 성장기 (Hero's Journey 변형)

1막: 비극적 출생 (아버지 살해) → 몽골 성장 → 강남칠괴에게 무공 전수
2막: 중원 진입 → 황용 만남 → 연속 기연 (홍칠공→강룡십팔장, 복사의 보혈)
3막: 정사 갈등 → 양강/완안홍렬과 대결 → 양양 수성
```

- **성장 곡선**: 느린 시작(우직한 자질) → 기연으로 계단식 도약 → 후반 안정
- **기연 패턴**: 우연+성품 보상형 (성실함이 기연을 부름)
- **악역**: 양강(소악역/거울상) → 구양봉(중악역/순수 무력) → 완안홍렬(대악역/정치적)

#### 신조협려 (도가적 협객 - 양과)

```
구조: 반항아의 고난과 사랑 (Romance + Rebellion)

1막: 아버지의 원수 곽정 아래 성장 → 고묘파 입문 → 소용녀와 사랑
2막: 금지된 사랑으로 강호 방랑 → 팔 절단(최대 위기) → 16년 이별
3막: 독자적 무공 창시(암연소혼장) → 재회 → 신조대협
```

- **성장 곡선**: 재능형 빠른 시작 → 팔 절단으로 급락 → 고통 속 독창적 성장
- **기연 패턴**: 역경 보상형 (고통이 클수록 큰 기연)
- **핵심**: 불같은 외골수 성격이 기연의 방향을 결정

#### 의천도룡기 (불가적 협객 - 장무기)

```
구조: 기연 집중형 상승 스토리

1막: 부모의 비극적 죽음 → 한독 중독 → 방랑
2막: 구양진경 습득 → 건곤대나이신공(며칠 만에 완성) → 명교 교주
3막: 육대문파 갈등 조정 → 의천검/도룡도의 비밀 → 은거
```

- **성장 곡선**: 기연 집중 폭발형 (단기간 최정상)
- **기연 패턴**: 누적 보상형 (어린 시절 고통의 대가)
- **특징**: 우유부단한 성격 - 원수에게도 자비 (불가적)

#### 천룡팔부 (불교적 비극/다주인공)

```
구조: 3주인공 병렬 → 수렴 구조

소봉 라인: 정의로운 강자 → 출생의 비밀(거란인) → 정체성 비극
허죽 라인: 소림 승려 → 우연한 기연 연속 → 강호 입문
단예 라인: 문약한 귀공자 → 기연 습득 → 성장

세 라인이 교차하며 최종 합류 → 공동의 적 대결
```

- **핵심**: 복잡한 인과관계, 형이상학적 주제
- **세력**: 대리국/소림/거란/서하/토번 (민족 간 갈등)
- **가장 난해한 김용 작품** - 불교적 업보와 숙명

#### 소오강호 (도가적 자유인 - 영호충)

```
구조: 정파 내부 부패 폭로 + 자유 추구

1막: 화산파 대제자 → 사부 위선 발견
2막: 강호 방랑 → 정사 경계 허물기 → 동방불패/일월신교
3막: 정파의 위선 vs 사파의 솔직함 → 진정한 자유 획득
```

- **핵심**: 정파≠선, 사파≠악이라는 전복
- **악역**: 악보인(사부/위선자) - 정파 내부의 적이 진짜 악역

#### 녹정기 (반영웅/피카레스크 - 위소보)

```
구조: 석세스 스토리 + 역사 대하소설

1막: 기루 출신 소년 → 황궁 잠입 → 강희제 친구
2막: 천지회 간부 겸직 → 이중 정체성으로 양쪽 활약
3막: 마누라 7명과 은거 → 해피엔딩
```

- **핵심**: 무공 없는 주인공 (잔머리와 화술로 해결)
- **김용의 자기 해체** - 무협소설의 공식을 의도적으로 파괴

### 1.3 김용 소설의 기연(奇緣) 구조

기연 = 주인공이 초월적 무공/지식을 얻는 우연한 계기 (출처: [나무위키 - 기연](https://namu.wiki/w/%EA%B8%B0%EC%97%B0))

**기연의 5대 유형**:
| 유형 | 설명 | 예시 |
|------|------|------|
| 비전서 발견 | 동굴/유적에서 비급 획득 | 장무기: 구양진경 |
| 고수 전수 | 은거 고수가 무공을 가르침 | 곽정: 홍칠공(강룡십팔장) |
| 약물/영단 | 특수 물질로 내공 급상승 | 곽정: 복사의 보혈 |
| 역경 돌파 | 절체절명 위기에서 깨달음 | 양과: 팔 절단 후 독자 무공 창시 |
| 유물 획득 | 전설적 무기/보물 | 장무기: 의천검/도룡도 |

**기연의 법칙**:
- 성품이 좋을수록 기연이 온다 (곽정의 우직함)
- 고통이 클수록 기연도 크다 (양과의 팔 절단)
- 기연은 3~5회 연속 배치된다 (한 번이 아닌 계단식)
- 주인공의 성격이 기연의 성격을 결정한다

### 1.4 세력 구조

**정파 (正派)**:
- 구파일방: 소림사, 무당파, 화산파, 곤륜파, 점창파, 공동파, 운산파, 청성파, 아미파
- 오대세가: 남궁세가, 모용세가, 하북팽가, 사천당문, 황보세가
- 무림맹 (구파일방의 연맹체)
- "북숭소림, 남존무당(北崇少林, 南尊武當)" - 소림과 무당이 양대 산맥

**사파 (邪派) / 마교**:
- 명교 (의천도룡기)
- 일월신교 (소오강호) - 동방불패
- 천마신교 (한국 무협 웹소설 기본 설정)
- 혈교, 배교, 새외무림

**김용의 전복**: 소림사도 타락하여 "사파와 구분 안 가는" 행태 (출처: [나무위키 - 소림사](https://namu.wiki/w/%EC%86%8C%EB%A6%BC%EC%82%AC(%EB%AC%B4%ED%98%91)))

출처: [나무위키 - 구파일방](https://namu.wiki/w/%EA%B5%AC%ED%8C%8C%EC%9D%BC%EB%B0%A9), [나무위키 - 무림세가](https://namu.wiki/w/%EB%AC%B4%EB%A6%BC%EC%84%B8%EA%B0%80)

## 2. 인기 웹소설들의 이야기 구조

### 2.1 나 혼자만 레벨업 (Solo Leveling)

출처: [위키백과](https://ko.wikipedia.org/wiki/%EB%82%98_%ED%98%BC%EC%9E%90%EB%A7%8C_%EB%A0%88%EB%B2%A8%EC%97%85), [나무위키](https://namu.wiki/w/%EB%82%98%20%ED%98%BC%EC%9E%90%EB%A7%8C%20%EB%A0%88%EB%B2%A8%EC%97%85)

**기본 구조**: E급 최약체 → "시스템" 획득 → 인류 최강

**아크 구조 (에스컬레이션 패턴)**:
```
아크 1: 각성 - 카르테논 신전 (이중 던전, 시스템 획득)
아크 2: 성장 - 합정역, VS 황동석, VS 강태식 (로컬 위협)
아크 3: 인정 - 백호 길드 스카우트, 전직 퀘스트 (사회적 위치 상승)
아크 4: 시련 - 레드 게이트, 악마성 1편 (팀 위기)
아크 5: 증명 - 헌터스 길드 A급 게이트, 협회 재검사 (S급 인정)
아크 6: 클라이맥스 - 악마성 2편, 제주도 레이드 (시즌 1 피크)
아크 7: 확장 - 일본편, 해외 헌터들 (세계관 확장)
아크 8: 최종 - 군주들과의 전쟁, 안타레스 결전
```

**보스전 에스컬레이션**:
| 단계 | 보스 | 스케일 |
|------|------|--------|
| D급 던전 | 일반 마수 | 개인 |
| C~B급 | 보스 마수 | 파티 |
| A급 | 아크 리치, 악마의 성 | 길드 |
| S급 | 개미왕 베르 (제주도) | 국가 |
| 군주급 | 바란, 안타레스 | 세계/차원 |

**긴장도 곡선**: 산맥형 (상승→해결→더 높은 위기 반복)

**파워 인플레이션 관리법**:
- "나만 레벨업 가능" 설정으로 다른 캐릭터와의 격차를 구조적으로 정당화
- 적의 스케일을 국내→해외→차원으로 확대
- 그림자 군단 시스템으로 과거 보스를 아군으로 전환 (보스 재활용)

**군주 목록** (최종 악역 계층):
- 아스본 (망자들의 왕/그림자 군주) - 주인공의 전임자
- 안타레스 (광룡들의 왕/파멸의 군주) - 최종 보스
- 바란 (악마들의 왕/백염의 군주)
- 레기아 (거인들의 왕)
- 퀘레샤 (벌레들의 왕/역병의 군주)

### 2.2 전지적 독자 시점 (Omniscient Reader's Viewpoint)

출처: [위키백과](https://ko.wikipedia.org/wiki/%EC%A0%84%EC%A7%80%EC%A0%81_%EB%8F%85%EC%9E%90_%EC%8B%9C%EC%A0%90), [나무위키](https://namu.wiki/w/%EC%A0%84%EC%A7%80%EC%A0%81%20%EB%8F%85%EC%9E%90%20%EC%8B%9C%EC%A0%90), [Postype 분석](https://www.postype.com/@paby1123/post/9823873)

**기본 구조**: 주인공이 읽던 소설이 현실이 됨 → 소설 지식으로 생존

**5부 구성** (총 551화):
```
1부: 서울 스타 스트림 (~시나리오 초반)
  - 지하철 정지 → 도깨비 출현 → 메인 시나리오 시작
  - 멸살법의 줄거리를 아는 주인공의 정보 우위 활용

2부: 중반 시나리오 확장
  - 성좌/도깨비 시스템 심화
  - 파티 구성과 세력 형성

3부: 세계관 확장
  - 다른 세계선과의 접촉
  - 시나리오의 본질에 대한 의문

4부: 진실 추적
  - 멸살법의 작가 정체
  - 이야기와 현실의 경계 해체

5부: 최종편
  - 메타내러티브 결말
  - "독자"와 "이야기"의 관계에 대한 철학적 결론
```

**시나리오 시스템의 구조적 혁신**:
- 메인 시나리오 + 서브 시나리오 + 히든 시나리오 3층 구조
- 시나리오 = 작중 "시련"이자 "이야기의 법칙"
- 시나리오에서 추방된 존재는 설화가 무너져 소멸

**에피소드 1 구조 분석**:
```
도입: 지하철 정지 (일상 파괴)
  → 원인 미공개 (서스펜스)
  → 도깨비 출현, 사람들 죽음
  → 메인 시나리오 퀘스트 부여

절정: 김남운과의 전투
  → 새로운 스킬 획득 (위기 중 보상)
  → 주인공 기개로 상황 역전

결말 훅: 더 큰 위기 (아포칼립스가 된 서울)
```

**반전 배치 패턴**:
1. "알고 있던 줄거리"가 틀어지는 순간 (정보 우위 상실)
2. 캐릭터가 원작과 다르게 행동 (나비효과)
3. 메타적 반전 (소설 밖의 진실)
4. 각 부 마지막에 세계관 재정의급 반전

### 2.3 한국 웹소설 공통 아크 구조

출처: [나무위키 - 무협 웹소설](https://namu.wiki/w/%EB%AC%B4%ED%98%91%20%EC%9B%B9%EC%86%8C%EC%84%A4), [나무위키 - 현대 판타지](https://namu.wiki/w/%ED%98%84%EB%8C%80%20%ED%8C%90%ED%83%80%EC%A7%80), [SlowNews](https://slownews.kr/74919)

**핵심 문법 (2010년대 중반 이후)**:
- **회빙환(回氷還)**: 회귀/빙의/환생 - 주인공이 정보 우위를 가진 채 재시작
- **사이다**: 답답함을 시원하게 해소하는 장면 (카타르시스)
- **망나니물**: 처음엔 쓰레기/바보 취급 → 각성 후 압도적 실력 공개

**현대판타지/헌터물 공통 아크**:
```
시즌 1 (30~50화): 각성 + 로컬 증명
  아크 1 (1~10화): 각성/시스템 획득 + 첫 던전
  아크 2 (11~20화): 첫 위기 + 사이다 장면
  아크 3 (21~30화): 사회적 인정 (길드/랭킹)
  아크 4 (31~50화): 시즌 보스 + 클라이맥스

시즌 2 (30~50화): 세력 확장
  아크 5~6: 국내 대형 사건
  아크 7~8: 새로운 세력/적 등장

시즌 3 (30~50화): 세계 무대
  아크 9~10: 해외 진출 or 차원 확장
  아크 11~12: 최종 대결

에필로그/외전: 일상 복귀 + 팬서비스
```

**무협 웹소설 공통 아크**:
```
시즌 1: 문파 입문 + 초기 성장
  아크 1: 출생의 비밀/회귀 → 무공 기반 마련
  아크 2: 문파 내 경쟁 (비무대회)
  아크 3: 외부 세력 침입 → 첫 실전

시즌 2: 강호 진출
  아크 4: 무림맹/대회 참가
  아크 5: 사파/마교와 갈등
  아크 6: 기연 획득 (비급/고수)

시즌 3: 무림 재편
  아크 7~8: 정사대전
  아크 9: 천하제일 달성 or 은거
```

## 4. 악역 공식

### 4.1 TV Tropes의 악역 에스컬레이션 체계

출처: [Sorting Algorithm of Evil](https://tvtropes.org/pmwiki/pmwiki.php/Main/SortingAlgorithmOfEvil), [Starter Villain](https://tvtropes.org/pmwiki/pmwiki.php/Main/StarterVillain), [Arc Villain](https://tvtropes.org/pmwiki/pmwiki.php/Main/ArcVillain), [Disc-One Final Boss](https://tvtropes.org/pmwiki/pmwiki.php/Main/DiscOneFinalBoss)

**Sorting Algorithm of Evil (악의 정렬 알고리즘)**:
> "첫 번째로 만나는 악역이 가장 약하고, 마지막이 가장 강하다."

```
계층 구조:

1. Starter Villain (소악역/스타터 빌런)
   - 히어로가 처음 싸우는 상대
   - Sorting Algorithm의 최하위
   - 역할: 주인공 능력 소개 + 독자 카타르시스 첫 맛

2. Arc Villain (아크 빌런/중악역)
   - 한 스토리 아크의 메인 적
   - 중간 단계의 사다리
   - 여러 명이 순차 등장

3. Disc-One Final Boss (가짜 최종보스)
   - "이게 최종보스인 줄 알았는데..." 반전
   - Big Bad Shuffle: 이런 반전이 여러 번 반복

4. Big Bad (대악역/최종보스)
   - 이야기 전체의 궁극적 적대자
   - 모든 갈등의 근원

5. The Man Behind the Man (흑막)
   - Big Bad 뒤의 진짜 배후
   - 최종 반전용
```

**변형 트로프**:
- **Not-So-Harmless Villain**: 약해 보이던 초기 악역이 사실 강했음
- **Snowballing Threat**: 약한 악역이 급격히 성장
- **Villain Decay**: 반대로 강했던 악역이 점점 약해짐
- **Make Way for the New Villains**: 새 악역이 기존 악역을 제거하며 등장
- **Rule of Escalating Threat**: 위협은 반드시 에스컬레이션해야 함

### 4.2 한국 웹소설의 악역 유형

출처: [디지털투데이](https://www.digitaltoday.co.kr/news/articleView.html?idxno=533572), [리디 분석](https://ridicorp.com/story/pr-villain/), [egowriting](https://egowriting.com/blog_sub.php?wr_id=42)

**매력적인 악역의 3대 조건**:
1. **카리스마**: 존재감과 위압감이 있는가
2. **이해 가능한 과거**: 독자가 공감할 수 있는 동기가 있는가
3. **큰 문제 제기**: 인간의 악/불의에 대한 본질적 질문을 던지는가

**악역 만들기 3원칙**:
1. **흑백논리 경계** - 순수악은 불필요. 악역에게도 긍정적 속성(지능, 매력, 능력) 부여
2. **캐릭터 편애 금지** - 주인공만 선하고 악역만 비열한 극단적 대비 지양
3. **내적 동기 부여** - 악역의 행동에 나름의 사연과 논리 존재

**미형 악역(美形悪役)** (출처: [나무위키 - 미형 악역](https://namu.wiki/w/%EB%AF%B8%ED%98%95%20%EC%95%85%EC%97%AD)):
- 고전~현대까지 생명력이 끊이지 않는 캐릭터 유형
- 최소 중간 이상의 인기 보장
- 악역 설정에 신경 썼다는 증거 = 작품 흥행의 블루칩

**악역 빙의물 트렌드** (출처: [Medium](https://stelland.medium.com/why-do-so-many-korean-webnovels-make-the-villainess-the-heroine-3f09b195da08)):
- 현실의 주인공이 소설 속 악역으로 빙의
- 납작한 악역의 삶에 개입하여 운명 개척
- 특히 로맨스판타지에서 "악녀" 키워드 지속 인기

### 4.3 악역 등장 타이밍과 교체 패턴

```
회차 기준 (100화 기준):

1~10화:   소악역 등장 (첫 사이다용)
           → 주인공 능력 과시 + 독자 확보

10~30화:  중악역 1 등장 (조직/세력 배경)
           → 주인공의 한계 시험

30~50화:  중악역 1 처리 + 대악역 그림자 노출
           → 세계관 확장의 신호탄

50~70화:  중악역 2 등장 (대악역의 부하/관련자)
           → 더 큰 음모 암시

70~90화:  대악역 본격 등장 + 최대 위기
           → 정면 대결 or 패배 후 성장

90~100화: 최종 결전
           → 새 시즌의 더 큰 악역 암시

핵심 법칙: 악역 교체 시 반드시 "새 악역이 이전 악역보다 강해 보이는 장면" 삽입
```

### 5.1 웹소설 조연 유형

- 정신병(집착/약점)의 강약을 조절하고 조합
- 그 특성이 "누구에게 열리고 닫히는지"만 설정하면 캐릭터 뽑힘

| **라이벌** | 경쟁 동기, 긴장감 | 양강(사조), 차해원(나혼렙) | 초반부터 지속 |
| **코미디릴리프** | 긴장 해소, 분위기 전환 | 위소보식 캐릭터 | 전체 배치 |
| **정보원** | 세계관 설명, 복선 투척 | 안내자/점술사 유형 | 아크 시작마다 |
| **충성파/부하** | 주인공 위상 과시, 전력 보충 | 그림자 군단(나혼렙) | 중반 이후 |
| **비극적 동료** | 감정적 동기 부여 (사망 플래그) | 시즌 중반 합류 후 희생 | 클라이맥스 직전 |
| **히로인/러브라인** | 감정적 앵커, 일상 파트 | 차해원, 소용녀 | 초반 |
| **귀인** | 위기 시 구원, 연결고리 | 노고수/은거자 유형 | 위기 직전 |

- "제일 눈에 띄는 특징 한두 가지만 묘사하고 넘기는 것이 깔끔"
- 묘사는 양이 아니라 방법 - "두 가지 기능"을 해야 함 (외모 전달 + 분위기/성격 암시)
- 특질 나열법: 머리→이마→눈빛→턱선→어깨 순서로 집어서 묘사

**3가지 묘사 방식**:
1. **직접 묘사**: 서술자가 직접 외모를 나열
2. **타인의 입을 통한 묘사**: 다른 캐릭터의 반응으로 외모 암시
3. **행동+외모 혼합**: 옷차림, 태도, 행동과 함께 외모 서술

### 6.2 캐릭터 유형별 외모 클리셰

**주인공**:
| 장르 | 클리셰 외모 | 왜 이렇게? |
|------|------------|-----------|
| 현대판타지 | "평범한 외모" (but 실제론 미형) | 동일시 + 갭 모에 |
| 무협 | 수려한 용모 or 투박하지만 강인한 인상 | 영웅 코드 |
| 회귀물 | 처음엔 초라 → 성장과 함께 외모 변화 | 성장 시각화 |
| 로판 | 비현실적 미모 상세 묘사 | 판타지 충족 |

**악역**:
| 유형 | 외모 코드 | 기능 |
|------|----------|------|
| 미형 악역 | 잘생김 + 차가운 눈빛 | 위험한 매력 = 긴장감 |
| 위압형 악역 | 거대한 체격, 흉터, 압도적 존재감 | 물리적 위협 전달 |
| 지략형 악역 | 마른 체격, 날카로운 눈 | 지적 위협 전달 |
| 광기형 악역 | 비정상적 미소, 뒤틀린 표정 | 예측불가 위협 |

**히로인**:
| 유형 | 외모 코드 | 대표 머리/눈 |
|------|----------|-------------|
| 순정파 | 단아하고 깨끗한 인상 | 흑발/갈색 + 맑은 눈 |
| 쿨뷰티 | 차갑고 날카로운 인상 | 은발/백발 + 날카로운 눈매 |
| 활발형 | 밝고 건강한 인상 | 적갈색/금발 + 큰 눈 |
| 미스터리 | 신비롭고 읽을 수 없는 분위기 | 보라/은색 + 이색동자 |

### 6.3 "미인 그려놓고 평범하다고 우기기" 클리셰

출처: [나무위키](https://namu.wiki/w/%EB%AF%B8%EC%9D%B8%20%EA%B7%B8%EB%A0%A4%EB%86%93%EA%B3%A0%20%ED%8F%89%EB%B2%94%ED%95%98%EB%8B%A4%EA%B3%A0%20%EC%9A%B0%EA%B8%B0%EA%B8%B0)

- 주인공 외모를 "평범하다"고 서술하면서 주변 반응은 미형 반응

### 7.5 파워 인플레이션 관리법

출처: [Andrew Rowe - Writing Progression Fantasy](https://andrewkrowe.wordpress.com/2019/03/02/writing-progression-fantasy/), [Andrew Rowe - Distinctions](https://andrewkrowe.wordpress.com/2022/11/04/distinctions-in-progression-fantasy-styles/), [Royal Road 포럼](https://www.royalroad.com/forums/thread/130646)

**Progression Fantasy의 파워 시스템 유형**:

| 유형 | 설명 | 예시 |
|------|------|------|
| Numeric Levels | 숫자 기반 레벨 (1~99 등) | 나혼렙, LitRPG 류 |
| Titled Levels | 이름 붙은 단계 | Cradle (Copper→Iron→Jade→Gold) |
| Relative Progression | 비교를 통한 성장 표현 | Mother of Learning |

**파워 차이 설계**:
- **큰 차이** (10x+): 솔로 플레이 적합, 파티 무의미화 위험
- **작은 차이** (1.5~2x): 하위 레벨도 전략으로 경쟁 가능 (Arcane Ascension)
- **특화형**: 캐릭터마다 다른 능력치 성장 → 레벨 격차에도 역할 유지

**6가지 관리 원칙**:
1. **성장은 점진적이어야** - Level 1→99 한 권에 달성하면 실패
2. **새 능력 > 단순 수치 증가** - "새로운 스킬 추가가 스탯 증가보다 흥미롭다"
3. **악역도 성장한다** - 주인공만 강해지면 긴장감 소멸
4. **사회적 영향 반영** - 순간이동 능력이 생기면 세계도 바뀌어야
5. **약점 즉시 해결 금지** - 장기적 약점 극복이 만족감 극대화
6. **선택이 있어야** - "합리적으로 다른 선택을 할 수 있어야 독자가 관여"

**진행 속도 선택지**:
```
[빠른 진행] (Solo Leveling 스타일)
  - 주인공이 기록적 속도로 성장
  - 독자 만족: 무쌍 카타르시스
  - 위험: 긴장감 유지 어려움

[보통 진행] (Mother of Learning 스타일)
  - 약간 빠르지만 합리적 범위
  - 독자 만족: 성장의 실감
  - 추천: 가장 안전한 선택

[느린 진행] (Forge of Destiny 스타일)
  - 현실적 속도의 성장
  - 독자 만족: 깊이와 몰입
  - 위험: 지루함 가능
```

**치트 유형** (빠른 성장의 정당화):
- **직접형**: 성장 배율 20배 (Iron Prince의 S-rank growth)
- **간접형**: 시간 압축 훈련실, 타임루프
- **구조형**: 유일한 클래스/스킬 보유

### 7.6 시즌제 구조 모델

```
모델 A: 30화 시즌 (총 3~5시즌)
  시즌당: 3개 소아크 + 1개 시즌 피크 보스전
  시즌 간: 2~3화 휴식 에피소드

모델 B: 50화 시즌 (총 2~3시즌)
  시즌당: 5개 소아크 + 2개 중아크 + 1개 시즌 클라이맥스
  시즌 간: 시간 스킵 or 세계관 전환

모델 C: 100화 단위 대서사 (총 200~500화)
  10화 단위 소아크
  50화 단위 중아크 (시즌 대체)
  절반 지점에 최대 반전
  마지막 50화: 최종 전개

나혼렙: 모델 C (약 270화 완결)
전독시: 모델 C (551화 완결, 5부 구성)
화산귀환: 모델 B (시즌제 장기 연재)
```

### 8.1 미결 #1: "왜 강해지냐" — 능력 기원 공식

#### A. 작품별 기원 메커니즘 요약

#### B. 기원 공식의 3가지 패턴

```
패턴 1: "설계된 그릇" (Designed Vessel)
  → 누군가 의도적으로 주인공을 만듦/선택함
  → 예: 나혼렙(아쉬본), 주술회전(켄자쿠), 도쿄구울(카노)
  → 장점: "왜?"가 중장기 떡밥이 됨
  → hunter-reborn 적합도: ★★★★★ (포식 능력의 기원 = 누가 설계했나?)

패턴 2: "우연한 사고" (Accidental Symbiosis)
  → 예정에 없던 사고로 능력 획득
  → 예: 기생수(이어폰), 도쿄구울(장기이식), 전슬라(전생 시 조건 우연 충족)
  → 장점: 즉시 공개 가능, 스토리가 "적응"에 집중
  → hunter-reborn 적합도: ★★★☆☆

패턴 3: "메타적 자격" (Meta Qualification)
  → 주인공의 존재론적 위치가 능력의 원천
  → 예: 전독시(유일한 독자), 괴담출근(오타쿠 지식)
  → 장점: 작품의 주제의식과 능력이 직결
  → hunter-reborn 적합도: ★★☆☆☆ (장르 결이 다름)
```

### 8.2 미결 #2: "어떻게 강해지냐" — 성장 시스템 공식

#### A. 나혼렙 (Solo Leveling) — 가장 직접적 레퍼런스

**스탯 시스템**:
| 스탯 | 영문 | 효과 |
|------|------|------|
| 근력 | Strength | 물리 공격력, 적재량, 중무기 운용 |
| 민첩 | Agility | 이동/공격 속도, 회피, 동체시력, 전투 선제권 |
| 체력 | Vitality | HP 총량, 물리 방어, 회복 속도, 상태이상 저항 |
| 지능 | Intelligence | MP 총량, 마법 데미지, 스킬 효과, 정신 저항 |
| 감각 | Perception | 적 탐지, 위험 감지, 명중률, 움직임 예측 |

**레벨업 메커니즘**:
- 레벨업 시 **5 스탯 포인트** 획득, 자유 배분
- 경험치 획득: 몬스터 처치(고랭크=고보상) + 퀘스트 완료 + 던전 클리어 + 보스 처치
- EXP 요구량 **지수적 증가** — 초반 빠른 레벨업, 후반 극적 둔화

**레벨 구간별 전투력**:
| 레벨 | 헌터 등급 환산 |
|------|--------------|
| 1-20 | E급 이하 |
| 21-50 | D급 (그림자 추출 해금) |
| 51-80 | 대부분의 S급 초과 |
| 81-100 | 인류 최강권 |
| 100+ | 인간 한계 초월 |

**일일 퀘스트 (강제 훈련)**:
- 팔굽혀펴기 100회, 윗몸일으키기 100회, 스쿼트 100회, 10km 달리기
- 미완료 시 "벌칙 구간(Penalty Zone)" 차원 소환 → 생존 전투
- 설계 의도: 전투 없는 날에도 성장 강제

**스킬 획득 방식**:
- 시스템 보상 (퀘스트/던전 클리어 시 자동 부여)
- 칭호 달성 (특정 업적 완수 시 영구 패시브)
- 계승 (아쉬본의 군주 능력 후반 전수: 군주의 권능/지배자의 손)

**핵심 차별점**: 일반 헌터는 각성 시 능력이 **고정** — 아무리 노력해도 성장 불가. 성진우만 유일하게 RPG식 레벨업 가능.

#### B. 전슬라 (전생했더니 슬라임이었던 건에 대하여)

**스킬 등급 시스템**:
| 등급 | 설명 |
|------|------|
| (일반) 스킬 | 기본 능력 |
| 엑스트라 스킬 | 강화된 능력 |
| 유니크 스킬 | 독자적 고유 능력 (마왕 후보급) |
| 얼티밋 스킬 (궁극 능력) | 최상위. 하위 스킬 **전부 무효화**. 얼티밋끼리만 대항 가능 (세계의 절대 법칙) |

**성장 메커니즘**:
- **마력량(마소량)** = 전투력의 척도
- **이름 부여(Naming)**: 상위 존재가 하위 마물에게 이름을 주면 → 마물이 **진화** + 마력 급증. 대신 이름 부여자의 마력 일부 소모
- **스킬 통합**: 대현자(→라파엘→시엘)가 복수 스킬을 합성/개변하여 상위 스킬 생성
- **포식/흡수**: 적을 포식 → 능력/스킬/소재 흡수 (포식자→폭풍대용→허공지신)
- **진화 조건**: 마왕으로의 진화 = 1만 명 이상의 인간 영혼 수확 + 세계의 소리 인정
- **마물 위협도 등급**: A~S급 위협도로 분류. 마왕종 = 재해급(Calamity Class) 이상

**대현자→라파엘→시엘 진화**:
- 대현자(유니크): 분석, 질의응답, 자동 전투 보조
- 지혜의 왕 라파엘(얼티밋): 스킬 통합/해석, 전투 AI
- 시엘(자아 각성): 독자적 판단, 스킬 창조, 사실상 전지전능 보조 AI

#### C. 오버로드 — 클래스 + 레벨 캡 시스템

**레벨 시스템**:
- 레벨 캡: **100** (+여분 경험치 0.9레벨분)
- 직업(클래스) 시스템: 직업당 **최대 15레벨**, 만렙 100이므로 **최소 7개** 직업 필요
- 종족 레벨 + 직업 레벨 합산 = 총 레벨

**마법 슬롯**:
- 1레벨당 3가지 마법 습득 가능 → 100레벨 매직 캐스터 = **300가지** 마법
- 캐시 아이템으로 **+100 슬롯** 추가 가능 → 최대 400
- 아인즈: 사령 마법 특화 + 오버로드(언데드 최상위종) = 718가지+ 마법

**성장 한계**: 게임 서비스 종료 후 이세계 전이 → 경험치 획득 불가 = 레벨업 불가. 기존 스펙으로만 싸움.

#### D. 더 게이머 (The Gamer) — 현실에 게임 시스템

**스탯 목록**:
| 스탯 | 효과 |
|------|------|
| STR (근력) | 물리력, 근육 |
| DEX (민첩) | 기민성, 정밀도 |
| VIT (체력) | HP, 물리 내성 |
| INT (지능) | MP 총량, 마법 위력 |
| WIS (지혜) | 인지력, 마법저항(10WIS당 1%), 마나재생(10WIS당 1%) |
| LUK (행운) | 확률 보정 |

**게이머 능력 핵심**:
- 잠을 자면 **HP/MP 전회복 + 상태이상 해제** (침대 회복)
- 일상 행동이 **스킬로 변환** (달리기 반복 → "달리기 Lv.1" 자동 생성)
- 스킬북 읽으면 **즉시 스킬 습득**
- 관련 스킬끼리 **합성 가능** → 상위 스킬 생성
- 최대 **8개 마법 합성** 가능 → 커스텀 마법 창조
- "반신급(Demigod Class)" 능력으로 분류 — 가장 희귀하고 강력한 등급

**레벨업**: 레벨업 시 스탯 포인트 획득, 자유 배분. 한지한은 INT 올인 → 대마법사 빌드.

**9대 스탯**:

**완벽 진화(Perfect Evolution)**: 종족+클래스+프로페션 모두 **동시에 99레벨** 달성 시 최상위 진화 선택지 해금

**스킬 등급**: 일반 < 비범(Uncommon) < 희귀(Rare) < 에픽(Epic) < 전설(Legendary) < 신화(Mythical) < 신성(Divine) — 등급이 높을수록 이름이 화려해짐

**클래스 예시**: Medium Warrior (밸런스형): 레벨당 STR+1, AGI+1, END+1, VIT+1, TOUGH+1, Free+1

### 8.3 미결 #3: "스킬을 어떻게 쓰냐" — 스킬 발동 메커니즘

#### A. 작품별 스킬 발동 방식

| 작품 | 발동 방식 | 자원 소비 | 쿨타임/제한 | 사거리 |
|------|----------|----------|------------|--------|
| **나혼렙** | 생각/의지 (시스템 인터페이스 통해). 그림자 추출 시 **"일어나라(Arise)"** 구두 선언 필수 | MP. 그림자 재소환 시 마나 소비 | 명시적 쿨타임 없음. MP 고갈이 사실상 제한 | 그림자 교환은 **거리 무제한** |
| **전슬라** | 생각/의지. 대현자(→시엘)가 자동 발동 보조. 일부 마법은 영창(구두 선언) | 마력량(마소). 얼티밋 스킬은 막대한 마력 소비 | 일부 스킬에 재사용 대기 있음. 포식 후 분석에 시간 필요 | 영역계 스킬은 광범위 |
| **주술회전** | 주력(저주 에너지) 조작. 술식 발동은 **수인(手印)이나 구두 선언** 으로 강화 가능. 축어(주문 영창)로 출력 증폭 | 주력. 과다 사용 시 신체에 부하 | 영역 전개는 **막대한 주력 소비** + 전개 유지에 지속 소비. 간이 영역은 제한적 | 영역 전개 = 일정 범위 내 **필중(必中)** |
| **HxH (넨)** | 오라(생명 에너지) 조작. 기본 4기: 텐(방어막)→렌(증폭)→제츠(소멸)→하츠(발현). **연습과 수련**으로 체득 | 오라(제한된 총량, 회복 가능) | 제약과 서약(制約と誓約): 스스로 제한을 걸수록 능력 강화 | 계통별 상이 (강화계=근접, 방출계=원거리) |
| **블리치** | 참백도 해방: 시해는 **해방 구호** 필수 (예: "걸어라, 신소") + 참백도 이름 부르기. 만해는 참백도 이름만 | 영압(靈壓). 사신의 영적 에너지 | 만해 유지에 영압 지속 소비. 귀도는 번호별 난이도 차이 | 참백도별 천차만별 |
| **괴담출근** | 문신 위에 손을 올려 **해당 괴담으로 변신**. 괴담 지식 기반 퍼즐 풀기 | 정신력/자아. 괴담에 잠식될수록 자아 소모 | 괴담 사용 시 정신 오염 누적. 과다 사용 = 자아 상실 | 괴담별 상이 |

#### B. 발동 방식의 3대 패턴

```
패턴 1: "시스템 인터페이스형" (System UI)
  → 생각만으로 발동. 시스템이 중개
  → 예: 나혼렙(상태창), 더 게이머(게이머 능력), Primal Hunter(시스템 알림)
  → 장점: 발동이 빠르고 직관적. 전투 묘사가 깔끔
  → 단점: 마법적 분위기 약함

패턴 2: "선언/영창형" (Incantation)
  → 구호, 이름, 주문을 외쳐야 발동
  → 예: 블리치(해방구호), 주술회전(축어), 나혼렙(Arise)
  → 장점: 연출적 카타르시스 극대화. 명장면 만들기 좋음
  → 단점: 전투 중 외칠 시간이 필요 → 전술적 약점이 됨

패턴 3: "신체 조작형" (Body Control)
  → 에너지를 신체로 조작/집중
  → 예: HxH(넨), 주술회전(주력), 도쿄구울(카구네)
  → 장점: 수련/성장과 직결. 숙련도에 따른 차이 표현 가능
  → 단점: 묘사가 추상적이 되기 쉬움
```

#### C. 자원 시스템 비교

|----------|------|---------|----------|

| 주력 | 주술회전 | 주술 사용 불가 + 신체 부하 | 시간 경과, 반전술식 |

### 8.4 미결 #4: "주인공이 물리적으로 어떻게 싸우냐" — 전투 스타일 공식

#### A. 작품별 주인공 전투 묘사

**1. 나혼렙 — 성진우**
- **무기**: 단검 이중 사용 (Dual Daggers)
- **전투 스타일**: 암살자형 — 속도와 정밀도 중심
  - "그림자 속에서 나타나 적이 반응하기 전에 제거"
  - Dagger Rush: 전방위에서 단검으로 연속 난타
  - Mutilation (절단): 단검 전용 크리티컬 강화 스킬
  - Advanced Dagger Arts: 단검 사용 시 데미지 **+33%**
- **보조 전술**: 그림자 군단 소환 → 잡몹은 군단이 처리, 성진우는 보스 직접 교전
- **이동기**: Shadow Exchange — 배치된 그림자 병사와 **즉시 위치 교환** (거리 무제한)
- **영역기**: Monarch's Domain — 영역 내 그림자 병사 **전 능력 50% 강화**
- **물리적 묘사**: "나타남 → 베기 → 사라짐" 반복. 체력과 민첩에 크게 의존. 후반부로 갈수록 군단 지휘관 성격 강화.

**2. 전슬라 — 리무루**
- **슬라임 형태**: 물리 공격 무효(물리 면역). 포식자로 적을 통째로 삼킴. 의사(pseudopod)로 타격.
- **인간형(시즈에 의태)**: 검술 사용 가능. 대부분의 전투를 인간형으로 수행.
- **핵심 전투 패턴**:
  1. 대현자/시엘이 **자동으로 최적 스킬 선택**
  2. 포식(흡수) → 분석 → 학습 → 즉석 대응
  3. 의태로 포식한 존재의 형태/능력 복제
  4. 궁극 스킬: 허공지신(虛空之神) — 시공간 조작급
- **물리적 묘사**: 후반으로 갈수록 물리 전투보다 **능력 대 능력** 싸움. 리무루 본인이 직접 때리는 장면은 줄어들고 전략/지휘 중심.

**3. 주술회전 — 이타도리 유지**
- **전투 스타일**: **맨주먹 격투** 중심 (초반에는 술식 없음)
  - 선천적 초인 체력 → 벽을 부수는 펀치력
  - **흑섬(Black Flash)**: 주력을 타격 순간 0.000001초 안에 물리 충격과 동시 충돌 → 공간 왜곡. 의도적 발동 불가능, 확률적.
  - 후반: 적혈조술(赤血操術) 각성 → 피를 무기화
  - 최종: 스쿠나의 술식 체현(斬撃) 사용 가능
- **물리적 묘사**: "격투기+주력 부스트". 주먹으로 치되, 주력이 관통력을 더함. 근접전 특화.

**4. HxH — 곤 프릭스**
- **전투 스타일**: 강화계 넨 사용자 + 맨주먹/킥 근접 격투
  - 쟈쟌켄(가위바위보): 구(주먹=강화계 펀치), 찌(가위=변화계 칼날), 빠(보=방출계 원거리)
  - 발동 시 **"최초는 구!"** 구두 선언
  - 제약과 서약: 선언 시간 동안 무방비 → 대신 위력 증폭
- **물리적 묘사**: 순수한 신체 능력 + 오라 부스트. 달리고, 뛰고, 때리는 격투. 전술과 심리전이 핵심.

**5. 블리치 — 쿠로사키 이치고**
- **전투 스타일**: **검술** 중심 (참백도 잔겟츠)
  - 시해: 거대한 칼 형태, 월아천충(月牙天衝) = 영압을 검기로 날림
  - 만해(천쇄참월): 칼이 작아지고 속도 극대화. 전투력 5~10배
  - 최종: 진정한 참백도 해방 → 허(Hollow)+퀸시+사신 능력 통합
- **4대 전투 방식**: 참(斬 = 검술), 권(拳 = 체술), 주(走 = 순보/이동기), 귀(鬼 = 귀도/마법). 이치고는 참+주 특화, 귀도는 거의 안 씀.
- **물리적 묘사**: 대검 휘두르기 + 고속 이동(순보). 베고, 막고, 피하는 검격전.

**6. 괴담출근 — 김솔음**
- **전투 스타일**: **비전투형 주인공** — 물리적 전투보다 **퍼즐/지식/규칙 기반 생존**
  - 괴담에는 "규칙(Rule)"이 있음 → 규칙을 파악하고 따르거나 역이용
  - 문신: 포식한 괴담의 자아를 문신으로 봉인 → 필요시 해당 괴담의 능력으로 변신
  - 쫄보 성격이지만 극한 상황에서 판단력/행동력 폭발
- **물리적 묘사**: 때리지 않는다. **도망치거나, 규칙을 풀거나, 변신한다**. SCP/로브코어 계열 — "이해하면 이기고, 모르면 죽는다."

#### B. 전투 스타일 유형 분류

```
유형 1: "근접 무기형" (Melee Weapon)
  → 검, 단검, 창 등 무기 운용
  → 예: 나혼렙(단검), 블리치(참백도)
  → 물리 묘사: 베기/찌르기/막기의 전투 안무

유형 2: "맨몸 격투형" (Unarmed Combat)
  → 주먹/발/체술 + 에너지 부스트
  → 예: 주술회전(주먹+주력), HxH(넨+격투), 기생수(미기 변형+신체능력)
  → 물리 묘사: 때리기/차기/잡기

유형 3: "변신/흡수형" (Transformation)
  → 적을 포식하거나 자신을 변형
  → 예: 전슬라(포식+의태), 괴담출근(괴담 변신), 도쿄구울(카구네)
  → 물리 묘사: 형태 변화 + 촉수/날개/갑옷 등 비인간적 전투

유형 4: "지휘관형" (Commander)
  → 소환물/부하를 지휘하며 본인은 후방 or 핵심타격
  → 예: 나혼렙 후반(그림자 군단), 전슬라 후반(템페스트 국가)
  → 물리 묘사: 명령 + 핵심 순간만 직접 교전

유형 5: "비전투/퍼즐형" (Puzzle/Knowledge)
  → 물리적으로 싸우지 않음. 지식과 규칙 파악으로 승리
  → 예: 괴담출근, 전독시 초반, 녹정기
  → 물리 묘사: 도주/은신/화술/규칙 이용
```

### 8.5 미결 #5: "일반 각성자는 어떻게 싸우냐" — 주인공과의 차이

#### A. 나혼렙: 일반 헌터 vs 성진우

**일반 헌터**:
- 각성 시 마력량 측정 → E~S급 **영구 고정** (A급까지는 측정기로 측정, S급은 한도 초과)
- 각성 시 역할 **자동 결정**: 탱커, 딜러(마법/물리), 힐러, 서포터 — 본인 선택 불가
- 성장 **불가** — "한 번 각성한 자는 본인이 얼마나 노력하든 그 능력이 성장하는 일은 없다"
- 파티/레이드 편성 필수 (탱커+딜러+힐러 조합)
- 장비 의존도 높음 (마법 강화 무기, 방어구)

**성진우와의 핵심 차이**:
| 항목 | 일반 헌터 | 성진우 |
|------|----------|--------|
| 성장 | 불가 (고정) | 무한 레벨업 |
| 역할 | 단일 고정 | 만능 (단검+마법+소환) |
| 파티 | 필수 | 솔로 + 그림자 군단 |
| 자원 | 마력 고정량 | MP 레벨업으로 증가 |
| 스킬 | 각성 시 부여된 것만 | 시스템 보상으로 계속 추가 |
| 등급 | 영구 고정 | E→S→군주급 무한 상승 |

#### B. 전슬라: 일반 모험가/마물 vs 리무루

**일반 모험가**:
- 모험가 길드에서 랭크 부여 (하위~S랭크)
- 마법사/검사/치유사 등 역할 분화
- 마력량이 전투력 척도 — 일반 모험가는 마력량이 제한적
- 스킬 습득: 수련/수행을 통해 점진적으로 → 매우 느림

**일반 마물**:
- 종족에 따른 고유 능력 (고블린=약함, 오크=중간, 오거=강함)
- 이름 없는 마물은 진화 불가 → **이름 부여** 받아야 진화
- 진화 경로가 종족별로 정해져 있음

**리무루와의 핵심 차이**:
- 리무루는 **포식으로 모든 능력을 흡수** — 종족/역할 제한 없음
- 대현자/시엘이 **자동 최적화** — 일반 존재는 수동 판단
- 이름 부여로 부하를 대량 진화 가능 — 국가 단위 전력 증강
- 얼티밋 스킬 = 하위 스킬 전부 무효화 → 일반 존재는 원천적으로 대항 불가

#### C. 주술회전: 일반 주술사 vs 이타도리

**주술사 등급 체계**:
| 등급 | 전투력 비유 | 수 |
|------|-----------|-----|
| 4급 | 목재 야구방망이로 여유 | 다수 |
| 3급 | 권총이 있으면 안심 | 상당수 |
| 2급 (준2급) | 산탄총으로 아슬아슬 | 적음 |
| 1급 (준1급) | 전차로도 불안 | 극소수 |
| 특급 | 논외/재해급 | **4명** |

**일반 주술사 전투**:
- 주력 조작 + 각자의 타고난 **술식(Innate Technique)** 1개
- 술식은 혈통으로 계승되거나 개인 고유
- 대부분 근접전 + 술식 조합 (주먹+술식, 무기+술식)
- 영역 전개는 1급 이상 극소수만 가능
- 주력 고갈 시 술식 사용 불가 → 신체 능력만으로 싸움

**이타도리와의 핵심 차이**:
- 초반: 술식 없이 **순수 체력**만으로 1급급 전투 (비정상적 신체 능력)
- 흑섬(Black Flash)을 실전에서 연속 발동 — 일반 주술사는 평생 한 번도 못 칠 수 있음
- 후반: 적혈조술 + 스쿠나의 참격 = **복수 술식 운용** (일반 주술사는 1개)
- 내구력: 스쿠나 손가락 20개를 수용하는 그릇 — 일반인은 1개만으로 사망

#### D. 괴담출근: 일반 직원 vs 김솔음

**일반 직원**:
- 괴담에 노출되면 규칙을 모르므로 당함 → 사망/잠식
- 잠식된 직원은 "괴담과 동화" → 종신직 (퇴사 불가)
- 괴이 대응 매뉴얼에 의존 — 매뉴얼에 없으면 대처 불가

**김솔음과의 핵심 차이**:
- 원작 지식(위키/팬덤 수준) → 매뉴얼에 없는 정보까지 파악
- 문신 시스템: 포식한 괴담의 자아를 봉인 → 능력으로 전환 (일반 직원은 잠식만 됨)
- 쫄보이지만 극한 상황의 판단력/행동력이 비정상적으로 뛰어남
- 일반 직원 = 규칙 기반 방어만 가능 vs 김솔음 = 규칙을 역이용하여 공격적 대응

### 8.6 hunter-reborn 적용을 위한 공식 추출 요약

#### 미결 #1 → "왜 강해지냐" 권장 공식
- **"설계된 그릇" 패턴** 채택 권장 (나혼렙/주술회전 스타일)
- 포식 능력의 기원 = 누군가(혹은 무언가)가 의도적으로 부여/선택
- "왜 이놈만?"은 중장기 떡밥으로 보류 → 중반 반전에서 공개

#### 미결 #2 → "어떻게 강해지냐" 권장 공식
- **나혼렙 + Primal Hunter 하이브리드**: 5~9개 스탯 + 레벨업 시 포인트 배분
- 포식 특화: 적 포식 시 **경험치 + 스킬/능력 흡수** (전슬라 포식자 참고)
- 성장 한계 돌파: 등급/진화 시스템 (Primal Hunter의 Grade 참고)

#### 미결 #3 → "스킬을 어떻게 쓰냐" 권장 공식
- **하이브리드**: 기본 스킬은 시스템 인터페이스형 (생각으로 발동) + 핵심 기술은 선언형 (연출용)
- 자원: MP 기반 + 포식 게이지 (이중 자원)
- 쿨타임: 등급별 차등 (일반 스킬=없음, 궁극기=긴 쿨타임 or 1일 1회)

#### 미결 #4 → "어떻게 싸우냐" 권장 공식
- **근접 무기형 + 변신/흡수형 혼합**
- 기본: 물리 무기(검/단검) + 포식한 능력으로 강화/변형
- 성장에 따라: 솔로 근접 → 포식체 소환 → 지휘관형으로 진화
- 물리 묘사의 핵심: "달려가서 베고, 먹고, 강해진다"

#### 미결 #5 → "일반 각성자 차이" 권장 공식
- **나혼렙 공식 직접 차용**: 일반 각성자 = 등급 고정 + 역할 고정 + 성장 불가
- 주인공만의 차별점: 포식에 의한 무한 성장 + 역할 자유 전환
- 긴장감 장치: 일반 각성자 파티가 주인공 없이는 위험에 빠지는 구도
