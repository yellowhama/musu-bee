Windows UAC OS Error 740 Finding

During local test verification, we identified a Windows-specific constraint:

Windows UAC OS Error 740: Running a global cargo test in an un-elevated command prompt fails on tests/r6_auto_update.rs. Windows UAC (User Account Control) automatically blocks execution of any binary with names containing keywords like "update" or "install" unless the shell is run with Administrator privileges (elevated). Because of this, integration testing on un-elevated Windows shells must be executed targeting specific suites (e.g., cargo test --test r7_peer_register) rather than executing a blanket test pass.
