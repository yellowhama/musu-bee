## §1 Thesis (sharpened post user decision)

**Primary wedge**: musu의 blue ocean = **agents가 multi-machine에 걸쳐 공유하는 영구 기억 (markdown wiki) + 사용자가 그걸 HTML로 즉시 흡수**. tailnet + paperclip은 baseline. agents 영구 기억 + 사용자 HTML 흡수 둘 다 paperclip이 못함.

**V23.5는 wedge를 demonstration-grade에서 differentiator-grade로 격상**:

- **W (primary thesis)**: agent-facing wiki를 HTML-render-on-read 가능하게. agents는 markdown 쓰고 markdown 읽음 (변경 0, token 효율 유지). musu-bee가 `/app/wiki/agent/<page_id>` 새 route에서 같은 markdown을 HTML로 렌더해 사용자에게 보여줌. Tariq 패턴 #14 적용.

- **S (docs prerequisite)**: SSOT_1PAGE을 4-layer 모델로 업데이트 (CoS = Layer 0).
