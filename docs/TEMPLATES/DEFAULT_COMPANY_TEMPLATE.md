# Default Company Template

This is the canonical operating baseline that MUSU should use when creating a new company workspace.

## Invariants

- one company-level operating goal
- product work separated from blocker work
- six default agent lanes
- supervisor maintenance enabled by default
- proof-backed board comment contract
- qualitative and audit reporting baked in

## Default lanes

- CEO
- CTO
- Founding Engineer
- QA Lead
- Chief of Staff
- Local Worker

## Default maintenance

- remediation normalize
- comment lint
- lint packetize

## Why this exists

MUSU already paid the cost of hardening these rules in live operation. New companies should inherit the baseline instead of rebuilding it manually.

## Product intent

When MUSU later gets a company-creation flow, this template should be the first selectable preset and the default unless the operator explicitly overrides it.
