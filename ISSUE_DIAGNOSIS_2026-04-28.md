## 오류 목록

1.  **[mgr-4060] Agent unavailable. Please try again later.**
    - **설명:** `mgr-4060` 에이전트에 연결할 수 없습니다. 일시적인 과부하 또는 에이전트 프로세스 자체의 문제일 수 있습니다.
    - **영향:** `mgr-4060` 에이전트를 사용하는 모든 태스크가 실패합니다.

2.  **[team_lead] Agent unavailable. Please try again later.**
    - **설명:** `team_lead` 에이전트를 사용할 수 없습니다. 위와 동일한 원인으로 추정됩니다.
    - **영향:** 리더 위임 및 조정과 관련된 태스크가 실패합니다.

3.  **[engineer] unhandled exception: unexpected internal error**
    - **설명:** `engineer` 에이전트 내부에서 처리되지 않은 예외가 발생했습니다. 코드 내의 버그일 가능성이 높습니다.
    - **영향:** 엔지니어링 관련 태스크가 예측 불가능하게 실패합니다. 상세한 로그 분석이 필요합니다.

4.  **[engineer] outer failure: semaphore boom**
    - **설명:** 세마포어 관련 오류로, 동시에 처리할 수 있는 작업 수를 초과했거나 리소스 경쟁 상태에서 발생했을 가능성이 있습니다.
    - **영향:** 동시성 제어가 필요한 엔지니어링 태스크가 실패합니다.

5.  **[engineer] channel_at_capacity**
    - **설명:** 에이전트 간 통신 채널이 가득 찼습니다. 메시지 처리 속도보다 생성 속도가 빠를 때 발생합니다.
    - **영향:** 에이전트 간의 통신이 지연되거나 실패하여 전체 워크플로우에 병목 현상을 유발합니다.

- 에이전트 가용성 문제를 해결하기 위해 노드 및 프로세스 상태를 점검합니다.
- 리소스 및 채널 용량 문제를 해결하기 위해 관련 설정을 검토하고 필요 시 확장합니다.
