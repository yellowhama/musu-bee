# MUSU Brand Token Map

- Cocoa Brown: `--musu-cocoa-brown` (`#2D1D19`)
- Musu Yellow: `--musu-yellow` (`#FFD166`)
- Off-White: `--musu-offwhite` (`#FDFCF0`)

Usage rule:
- Public surface components/pages must reference CSS vars (or semantic aliases mapped to these vars).
- Raw brand hex literals are disallowed outside `src/app/globals.css`.
