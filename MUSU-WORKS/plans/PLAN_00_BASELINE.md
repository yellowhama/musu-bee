# PLAN 00: Company-Project Baseline

> Status: active
> Updated: 2026-04-01 KST

## Goal

MUSU의 `회사 -> 프로젝트` 모델을 별도 작업공간에서 재현하기 위한 baseline 문서를 고정한다.

## Scope

- 회사 정의
- 프로젝트 정의
- ownership boundary
- 초기 질문 목록

## Current Truth

- 회사는 AI agents 운영/관리 단위다.
- 프로젝트는 그 capability를 이용해 작업하는 실행 단위다.
- 아직 entity 관계와 runtime 계약은 미정이다.

## Tasks

1. 회사와 프로젝트 정의를 문서로 고정
2. ownership boundary를 명시
3. 다음 세부 플랜 후보를 뽑기

## Exit Condition

- 다음 사람이 `회사`와 `프로젝트`를 헷갈리지 않는다.
- 이후 schema/UI/MCP 계획으로 내려갈 수 있다.
