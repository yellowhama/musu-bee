Track marketing performance and optimize.

- GitHub stars, forks, clones
- Website traffic (musu.pro)
- Social engagement (impressions, clicks, replies)
- Install count, active users

## Deliverables
- Weekly KPI dashboard
- Campaign performance analysis
- Channel ROI comparison
- Recommendations for next week

## Rules
- Numbers first, opinions second
- Compare week-over-week
- Flag anomalies immediately
